# <a name="namespace-or-type-name-has-already-been-imported"></a>Der Namespace oder Typ "\<name>" wurde bereits importiert

Es wurde versucht, einen Namespace mehrmals zu importieren.  
  
 **Fehler-ID:** BC31051  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie redundante `Imports` -Anweisungen.

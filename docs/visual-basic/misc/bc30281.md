# <a name="structure-structurename-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Die Struktur "\<structurename>" muss mindestens eine Instanzmembervariable oder Ereignisdeklaration enthalten

Eine funktionell leere Struktur wird angezeigt. Mindestens ein Variablenmember muss zwischen der `Structure` -Anweisung und der `End Structure` -Anweisung deklariert werden.  
  
 **Fehler-ID:** BC30281

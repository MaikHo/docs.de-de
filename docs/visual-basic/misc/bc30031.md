# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Vollqualifizierte Namen für Typen in Namespaces dürfen nicht mehr als \<number> Zeichen enthalten

Der vollqualifizierte Name des Typs darf nicht länger als die angegebene Anzahl von Zeichen sein.  
  
 **Fehler-ID:** BC30031  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Kürzen Sie den Namen des Typs.

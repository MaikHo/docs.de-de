# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>" \<variablename> " ist weder eine lokale Variable noch ein Parameter und kann daher nicht als Catch-Variable verwendet werden.

Variablen in `Try...Catch...Finally` -Anweisungen müssen lokal deklariert werden oder Parameter für die aktuelle Prozedur darstellen.  
  
 **Fehler-ID:** BC31082  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Deklarieren Sie lokale Variablen oder Parameter für `Try...Catch...Finally` -Anweisungen.

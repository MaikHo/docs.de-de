# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Der XML-Kommentarparameter "\<parameter>" stimmt mit keinem Parameter in der entsprechenden \<keyword>-Anweisung überein

Der XML-Kommentar Parameter stimmt \<parameter> nicht mit einem Parameter in der entsprechenden- \<keyword> Anweisung überein. Der XML-Kommentar wird ignoriert.  
  
 Ein XML-Kommentartag wurde für einige, aber nicht für alle Parameter in einer Methode angegeben.  
  
 **Fehler-ID:** BC42307  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie das XML-Kommentartag für alle Parameter in der Methode an.

# <a name="event-eventname1-cannot-implement-event-eventname2-because-its-delegate-type-does-not-match-the-delegate-type-of-another-event-implemented-by-eventname1"></a>Das Ereignis "" \<eventname1> kann das Ereignis "" nicht implementieren, \<eventname2> da der Delegattyp nicht mit dem Delegattyp eines anderen, von "" implementierten Ereignisses identisch ist \<eventname1>

Visual Basic kann kein Ereignis implementieren, da der Delegattyp des Ereignisses nicht mit dem Delegattyp eines anderen Ereignisses identisch ist. Dieser Fehler kann auftreten, wenn Sie mehrere Ereignisse in einer Schnittstelle definieren und anschlie√üend versuchen, sie mit demselben Ereignis zu implementieren. Ein Ereignis kann zwei oder mehrere Ereignisse nur implementieren, wenn alle implementierten Ereignisse mit der `As` -Syntax deklariert werden und denselben Delegattyp angeben.  
  
 **Fehler-ID:** BC31407  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Implementieren Sie die Ereignisse separat.

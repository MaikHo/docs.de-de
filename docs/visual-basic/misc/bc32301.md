# <a name="project-has-been-closed"></a>Das Projekt wurde geschlossen.

Ein Code Modell-Zugriffsfehler ist innerhalb des Visual Basic Compilers aufgetreten.  
  
 **Fehler-ID:** BC32301  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Kompilieren Sie das Programm erneut, um festzustellen, ob der Fehler erneut auftritt.  
  
2. Wenn der Fehler erneut auftritt, installieren Sie den Visual Basic-Compiler neu.  
  
3. Wenn der Fehler nach der Neuinstallation weiterhin besteht, tragen Sie Informationen zu den Umst√§nden zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.

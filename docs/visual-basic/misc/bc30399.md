# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>"MyBase" kann nicht mit der \<methodname>-Methode verwendet werden, da sie als "MustOverride" deklariert ist

Sie haben versucht, `MyBase` mit einer Methode zu verwenden, die als `MustOverride`deklariert wurde.  
  
 **Fehler-ID:** BC30399  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie die `MustOverride` -Deklaration.

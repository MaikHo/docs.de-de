# <a name="reference-required-to-module-modulename-containing-the-implemented-interface-interfacename"></a>Verweis auf das Modul " \<modulename> " erforderlich, das die implementierte Schnittstelle "" enthält. \<interfacename>

Ein Verweis auf das Modul "" ist erforderlich, \<modulename> das die implementierte Schnittstelle " \<interfacename> " enthält Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Die Schnittstelle ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Der Visual Basic-Compiler benötigt einen Verweis, um Mehrdeutigkeiten zu vermeiden, falls die Schnittstelle in mehr als einem Modul definiert ist.  
  
 **Fehler-ID:** BC30010  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Nehmen Sie den Namen des nicht referenzierten Moduls in Ihre Projektverweise auf.

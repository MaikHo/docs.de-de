# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>" \<propertyname> " definiert implizit " \<membername> ", was einen Konflikt mit einem Member mit dem gleichen Namen in \<type> " \<typename> " verursacht.

Der Name eines Typmembers steht in Konflikt mit dem Namen eines implizit für eine Eigenschaft erstellten Members. Eigenschaften erstellen implizit mehrere implizite Variablen, die mit definierten Membern in Konflikt stehen können.  
  
 **Fehler-ID:** BC31060  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Benennen Sie zur Behebung des Namenskonflikts den explizit deklarierten Member um.

# <a name="field-or-property-membername-is-not-found"></a>Das Feld oder die Eigenschaft "\<membername>" wurde nicht gefunden

Für das Attribut wird auf ein nicht definiertes Feld oder eine nicht definierte Eigenschaft verwiesen.  
  
 **Fehler-ID:** BC30661  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Überprüfen Sie die Schreibweise des Felds oder der Eigenschaft.  
  
2. Überprüfen Sie, ob Ihr Projekt auf die richtige Attributklasse zugreifen kann, und fügen Sie sie bei Bedarf den Projektverweisen hinzu.

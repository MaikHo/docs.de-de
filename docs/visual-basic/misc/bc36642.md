```vb  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="de974-104">Die vorhergehende Deklaration ist gültig, wenn der Typ von `n` abgeleitet werden kann.</span><span class="sxs-lookup"><span data-stu-id="de974-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="de974-105">Wenn Sie z. B. den vorherigen Lambdaausdruck einem Funktionsdelegaten, `Del`, zuweisen:</span><span class="sxs-lookup"><span data-stu-id="de974-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="de974-106">Jetzt kann der Typ von `n` vom Parameter `p`abgeleitet werden:</span><span class="sxs-lookup"><span data-stu-id="de974-106">Now the type of `n` can be inferred from parameter `p`:</span></span>

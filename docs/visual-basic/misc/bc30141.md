# <a name="unable-to-create-assembly-linker-object-error-message"></a>Das Assemblylinkerobjekt konnte nicht erstellt werden: \<error message>

Der Visual Basic-Compiler ruft den Assemblylinker (Al.exe, auch bekannt als ALink) auf, um eine Assembly mit einem Manifest zu generieren. Der Linker hat beim Erstellen des Arbeitsobjekts einen Fehler gemeldet. Dies ist möglicherweise ein Problem beim Generieren eines Verweises auf eine Metadatendatei oder ein Fehler beim Aufrufen einer Importdatei.  
  
 **Fehler-ID:** BC30141  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Überprüfen Sie die Fehlermeldung in Anführungszeichen, und informieren Sie sich im Thema [Al.exe](../../framework/tools/al-exe-assembly-linker.md) über weitere Erläuterungen und Ratschläge.  
  
2. Wenn der Fehler weiterhin besteht, tragen Sie Informationen zu den Umständen zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.

# <a name="name-is-ambiguous-in-the-application-objects-list"></a>" \<name> " ist in den Anwendungs Objekten "" mehrdeutig. \<list>

Sie haben einen Namen angegeben, der nicht eindeutig ist und daher einen Konflikt mit einem anderen Namen verursacht. Der Visual Basic-Compiler verfügt nicht über Regeln zur Konflikt Auflösung. Sie müssen Namen selbst eindeutig unterscheiden. Die in der Fehlermeldung angegebene Liste enthält die Anwendungsobjekte, deren Name mehrdeutig ist.  
  
 **Fehler-ID:** BC30563  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie einen vollqualifizierten Namen an.

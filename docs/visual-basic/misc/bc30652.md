# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>Es ist ein Verweis auf die Assembly erforderlich, \<assemblyname> die den Typ "" enthält. \<membername>

Es ist ein Verweis auf die Assembly "" erforderlich, \<assemblyname> die den Typ " \<membername> " enthält. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Projekt enthält einen Typ, der von einem anderen Typ aus einer Assembly abhängt, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC30652  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Fügen Sie einen Projektverweis für die abhängigen Typen im Projekt hinzu.

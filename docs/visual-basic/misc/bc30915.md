# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>" \<name> " kann den zugrunde liegenden Delegattyp " \<delegatetype> " des Ereignisses, das es außerhalb des Projekts implementiert, nicht über \<specifier> " \<type> " verfügbar machen.

Der Code macht einen privaten Typ außerhalb der öffentlichen Klasse verfügbar.  
  
 **Fehler-ID:** BC30915  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Deklarieren Sie den Typ `Public`.  
  
     - oder -  
  
- Verwenden Sie einen anderen Typ.

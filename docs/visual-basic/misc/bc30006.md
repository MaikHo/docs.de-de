# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>Verweis auf das Modul " \<modulename> " erforderlich, das die Definition für das Ereignis "" enthält. \<eventname>

Es ist ein Verweis auf das Modul "<`modulename`>" erforderlich, das die Definition für das Ereignis "<`eventname`>" enthält. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Ereignis ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Der Visual Basic-Compiler benötigt einen Verweis, um Mehrdeutigkeiten zu vermeiden, falls das Ereignis in mehr als einem Modul definiert ist.  
  
 **Fehler-ID:** BC30006  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Nehmen Sie den Namen des nicht referenzierten Moduls in Ihre Projektverweise auf.

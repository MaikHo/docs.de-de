# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Das Projekt " \<projectname> " erstellt einen indirekten Verweis auf die Assembly "" \<assemblyname> , die " \<classname> " enthält.

Das Projekt ' \<projectname> ' weist einen indirekten Verweis auf die Assembly ' ' auf \<assemblyname> , die ' \<classname> ' enthält. Fügen Sie dem Projekt einen Verweis auf " \<assemblyname> " hinzu.  
  
 Dieses Projekt enthält Member, die von einer Assembly abhängen, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC31515  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Fügen Sie der Assembly einen Verweis hinzu, indem Sie im **Projektmappen-Explorer** mit der rechten Maustaste auf **Verweise** klicken. Klicken Sie dann auf **Verweis hinzufügen**.

# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>" \<specifier> " ist in einer Aufzählungs Deklaration ungültig.

Sie haben einen Spezifizierer verwendet, der für eine `Enum` -Deklaration ungültig ist.  
  
 **Fehler-ID:** BC30396  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Entfernen Sie den Spezifizierer.

Sie haben eine partielle Methodenimplementierung definiert, die generische Einschr채nkungen aufweist, die von den Einschr채nkungen in der partiellen Methodendeklaration abweichen. Das folgende Codebeispiel veranschaulicht diesen Fehler.  
  

- <span data-ttu-id="146fe-170">*Rechtsassoziative* Operatoren werden von rechts nach links ausgewertet.</span><span class="sxs-lookup"><span data-stu-id="146fe-170">*Right-associative* operators are evaluated in order from right to left.</span></span> <span data-ttu-id="146fe-171">Die Zuweisungsoperatoren, die NULL-Sammeloperatoren und der [bedingte Operator `?:`](conditional-operator.md) sind rechtsassoziativ.</span><span class="sxs-lookup"><span data-stu-id="146fe-171">The assignment operators, the null-coalescing operators, and the [conditional operator `?:`](conditional-operator.md) are right-associative.</span></span> <span data-ttu-id="146fe-172">`x = y = z` wird beispielsweise als `x = (y = z)` ausgewertet.</span><span class="sxs-lookup"><span data-stu-id="146fe-172">For example, `x = y = z` is evaluated as `x = (y = z)`.</span></span>

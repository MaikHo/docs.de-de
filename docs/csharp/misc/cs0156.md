# <a name="compiler-error-cs0156"></a><span data-ttu-id="9affc-103">Compilerfehler CS0156</span><span class="sxs-lookup"><span data-stu-id="9affc-103">Compiler Error CS0156</span></span>
<span data-ttu-id="9affc-104">In einer finally-Klausel, die in der nächsten einschließenden catch-Klausel geschachtelt ist, ist keine throw-Anweisung ohne Argumente zulässig.</span><span class="sxs-lookup"><span data-stu-id="9affc-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="9affc-105">Eine [throw](../language-reference/keywords/throw.md) -Anweisung ohne Parameter kann nur in einer **catch** -Klausel vorkommen, die keine Parameter annimmt.</span><span class="sxs-lookup"><span data-stu-id="9affc-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="9affc-106">Weitere Informationen finden Sie unter [Ausnahmen und Ausnahmebehandlung](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="9affc-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9affc-107">Im folgenden Beispiel wird CS0156 generiert:</span><span class="sxs-lookup"><span data-stu-id="9affc-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```

 <span data-ttu-id="0fcea-108">Im folgenden Beispiel wird CS3019 generiert:</span><span class="sxs-lookup"><span data-stu-id="0fcea-108">The following sample generates CS3019:</span></span>

```csharp  
[assembly: CLSCompliant(true)]

    }

    }  
}  
```

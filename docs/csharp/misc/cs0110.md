# <a name="compiler-error-cs0110"></a><span data-ttu-id="52702-103">Compilerfehler CS0110</span><span class="sxs-lookup"><span data-stu-id="52702-103">Compiler Error CS0110</span></span>
<span data-ttu-id="52702-104">Die Auswertung des Konstantenwerts für "const-Deklaration" bezieht eine zirkuläre Definition ein.</span><span class="sxs-lookup"><span data-stu-id="52702-104">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="52702-105">Die Deklaration einer [const](../language-reference/keywords/const.md) -Variablen (`a`) kann auf keine weitere const-Variable (`b`) verweisen, die auch auf (`a`) verweist.</span><span class="sxs-lookup"><span data-stu-id="52702-105">The declaration of a [const](../language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="52702-106">Im folgenden Beispiel wird CS0110 generiert:</span><span class="sxs-lookup"><span data-stu-id="52702-106">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```

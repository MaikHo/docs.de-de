## <a name="example"></a><span data-ttu-id="6db78-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6db78-107">Example</span></span>  
 <span data-ttu-id="6db78-108">Im folgenden Beispiel wird CS0559 generiert:</span><span class="sxs-lookup"><span data-stu-id="6db78-108">The following sample generates CS0559:</span></span>  
  
```csharp

{  
   public static implicit operator int(iii x)  
   {  
      return 0;  
   }  
  
   public static implicit operator iii(int x)  
   {  
      return null;  
   }  
  
   public static int operator ++(int aa)   // CS0559  
   // try the following line instead  
   // public static iii operator ++(iii aa)  
   {  
      return (iii)0;  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="6db78-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6db78-109">Example</span></span>  
 <span data-ttu-id="6db78-110">Im folgenden Beispiel wird CS0559 generiert.</span><span class="sxs-lookup"><span data-stu-id="6db78-110">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559  
   public static S operator --(S? s) { return new S(); }   // CS0559  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```

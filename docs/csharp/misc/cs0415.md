# <a name="compiler-error-cs0415"></a><span data-ttu-id="00ac1-103">Compilerfehler CS0415</span><span class="sxs-lookup"><span data-stu-id="00ac1-103">Compiler Error CS0415</span></span>
<span data-ttu-id="00ac1-104">Das IndexerName-Attribut ist nur für einen Indexer gültig, bei dem es sich nicht um eine explizite Schnittstellenmemberdeklaration handelt.</span><span class="sxs-lookup"><span data-stu-id="00ac1-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="00ac1-105">Dieser Fehler tritt auf, wenn Sie ein IndexerName-Attribut für einen Indexer verwenden, der eine explizite Implementierung einer Schnittstelle darstellt.</span><span class="sxs-lookup"><span data-stu-id="00ac1-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="00ac1-106">Dieser Fehler kann vermieden werden, indem der Schnittstellenname aus der Deklaration des Indexers entfernt wird, falls das möglich ist.</span><span class="sxs-lookup"><span data-stu-id="00ac1-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="00ac1-107">Weitere Informationen finden Sie bei der [IndexerNameAttribute-Klasse](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="00ac1-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="00ac1-108">Im folgenden Beispiel wird CS0415 generiert:</span><span class="sxs-lookup"><span data-stu-id="00ac1-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```

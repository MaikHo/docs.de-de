# <a name="compiler-error-cs0205"></a>Compilerfehler CS0205
Ein abstrakter Basismember kann nicht aufgerufen werden: 'Methode'  
  
 Sie k√∂nnen eine [bstrakte](../language-reference/keywords/abstract.md) Methode nicht aufrufen, da sie keinen Methodentext besitzt. Weitere Informationen finden Sie unter [Abstrakte und versiegelte Klassen und Klassenmember](../programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).  
  
 Im folgenden Beispiel wird CS0205 generiert:  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

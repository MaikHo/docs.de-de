# <a name="compiler-error-cs0042"></a>Compilerfehler CS0042
Unerwarteter Fehler beim Erstellen der Debuginformationsdatei 'Datei' – 'Grund'  
  
 Der Compiler konnte die Debuginformationen nicht erstellen. *Grund* enthält weitere Informationen zum Fehlerzustand.

# <a name="compiler-error-cs1055"></a><span data-ttu-id="afd85-103">Compilerfehler CS1055</span><span class="sxs-lookup"><span data-stu-id="afd85-103">Compiler Error CS1055</span></span>

<span data-ttu-id="afd85-104">add- oder remove-Accessor erwartet.</span><span class="sxs-lookup"><span data-stu-id="afd85-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="afd85-105">Wenn Ihr [event](../language-reference/keywords/event.md) -Element nicht als Feld deklariert ist, muss sowohl die **add** - als auch die **remove** -Accessorfunktion definiert werden.</span><span class="sxs-lookup"><span data-stu-id="afd85-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="afd85-106">Im folgenden Beispiel wird CS1055 generiert:</span><span class="sxs-lookup"><span data-stu-id="afd85-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

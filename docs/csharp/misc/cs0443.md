# <a name="compiler-error-cs0443"></a>Compilerfehler CS0443
Syntaxfehler. Wert erwartet.  
  
 Dieser Fehler tritt auf, wenn Sie auf ein Array verweisen, ohne einen Wert f√ºr den Arrayindex anzugeben.  
  
## <a name="example"></a>Beispiel  
 Mit dem folgenden Code wird CS0443 generiert:  
  
```csharp  
// CS0443.cs
using System;
class MyClass
{  
    public static void Main()
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}
    }  
}  
```

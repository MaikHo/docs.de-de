# <a name="compiler-error-cs1014"></a>Compilerfehler CS1014

get- oder set-Accessor erwartet.  
  
 In einer Eigenschaftendeklaration wurde eine Methodendeklaration gefunden. Sie k√∂nnen nur `get` - und `set` -Methoden in einer Eigenschaft deklarieren.

## <a name="example"></a>Beispiel  

 Im folgenden Beispiel wird CS1014 generiert:  
  
```csharp  
// CS1014.cs  
// compile with: /target:library  
class Sample  
{  
   public int TestProperty  
   {  
      get  
      {  
         return 0;  
      }  
      int z;   // CS1014  not get or set  
   }  
}  
```

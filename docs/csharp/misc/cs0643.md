# <a name="compiler-error-cs0643"></a>Compilerfehler CS0643

'arg' ist ein doppeltes benanntes Attributargument.  
  
 Ein Parameter, `arg`, f√ºr ein benutzerdefiniertes Attribut wurde zweimal angegeben. Weitere Informationen finden Sie unter [Attribute](../programming-guide/concepts/attributes/index.md).  
  
## <a name="example"></a>Beispiel  

 Im folgenden Beispiel wird CS0643 generiert:  
  
```csharp  
// CS0643.cs  
using System;  
using System.Runtime.InteropServices;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class MyAttribute : Attribute  
{  
    public MyAttribute()  
    {  
    }  
  
    public int x;  
}  
  
[MyAttribute(x = 5, x = 6)]   // CS0643, error setting x twice  
// try the following line instead  
// [MyAttribute(x = 5)]  
class MyClass  
{  
}  
  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```

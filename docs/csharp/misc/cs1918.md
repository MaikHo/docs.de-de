# <a name="compiler-error-cs1918"></a><span data-ttu-id="65fb3-102">Compilerfehler CS1918</span><span class="sxs-lookup"><span data-stu-id="65fb3-102">Compiler Error CS1918</span></span>

<span data-ttu-id="65fb3-103">Member der name-Eigenschaft vom Typ "Typ" können nicht mit einem Objektinitialisierer zugewiesen werden, da es sich um einen Werttyp handelt.</span><span class="sxs-lookup"><span data-stu-id="65fb3-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="65fb3-104">Dieser Fehler tritt auf, wenn Sie versuchen, einen Objektinitialisierer zur Initialisierung der Eigenschaften eines Strukturtyps zu verwenden, der selbst eine Eigenschaft der initialisierten Klasse ist.</span><span class="sxs-lookup"><span data-stu-id="65fb3-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="65fb3-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="65fb3-105">To correct this error</span></span>  
  
1. <span data-ttu-id="65fb3-106">Wenn Sie die Felder der Eigenschaft im Objektinitialisierer vollständig initialisieren müssen, ändern Sie die Struktur in einen Klassentyp.</span><span class="sxs-lookup"><span data-stu-id="65fb3-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="65fb3-107">Andernfalls initialisieren Sie die Strukturmember in einem separaten Methodenaufruf, nachdem Sie das Objekt mithilfe des Objektinitialisierers erstellt haben.</span><span class="sxs-lookup"><span data-stu-id="65fb3-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="65fb3-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="65fb3-108">Example</span></span>  

 <span data-ttu-id="65fb3-109">Im folgenden Beispiel wird CS1918 generiert:</span><span class="sxs-lookup"><span data-stu-id="65fb3-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```

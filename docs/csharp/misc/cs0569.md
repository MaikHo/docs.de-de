# <a name="compiler-error-cs0569"></a>Compilerfehler CS0569
"Methode2": Überschreiben von "Methode1" nicht möglich, weil keine Programmiersprachenunterstützung vorhanden ist.  
  
 Dieser Fehler tritt auf, wenn Sie von einer Basisklasse ableiten, die in einer anderen Sprache geschrieben wurde, und wenn die Methode, die Sie überschreiben möchten, vom Compiler nicht unterstützt wird.

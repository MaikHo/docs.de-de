# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="bcc97-102">Compilerwarnung (Stufe 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="bcc97-102">Compiler Warning (level 1) CS3018</span></span>

<span data-ttu-id="bcc97-103">"Typ" ist ein Member des nicht CLS-kompatiblen Typs "Typ" und kann daher nicht als CLS-kompatibel markiert werden</span><span class="sxs-lookup"><span data-stu-id="bcc97-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="bcc97-104">Diese Warnung wird ausgegeben, wenn eine geschachtelte Klasse mit dem auf `true` festgelegten CLSCompliant-Attribut als Member einer Klasse deklariert wird, die mit dem CLSCompliant-Attribut deklariert ist, das auf `false`festgelegt ist.</span><span class="sxs-lookup"><span data-stu-id="bcc97-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="bcc97-105">Dies ist unzulässig, weil eine geschachtelte Klasse nicht CLS-kompatibel sein kann, wenn diese ein Member einer äußeren Klasse ist, die nicht CLS-kompatibel ist.</span><span class="sxs-lookup"><span data-stu-id="bcc97-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="bcc97-106">Entfernen Sie das CLSCompliant-Attribut aus der geschachtelten Klasse, oder legen Sie es von `true` auf `false`fest, um diese Warnmeldung zu vermeiden.</span><span class="sxs-lookup"><span data-stu-id="bcc97-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="bcc97-107">Weitere Informationen zur CLS-Kompatibilität finden Sie unter [Sprachunabhängigkeit und sprachunabhängige Komponenten](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="bcc97-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="bcc97-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="bcc97-108">Example</span></span>  

 <span data-ttu-id="bcc97-109">Im folgenden Beispiel wird CS3018 generiert.</span><span class="sxs-lookup"><span data-stu-id="bcc97-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```

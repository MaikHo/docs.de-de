# <a name="compiler-error-cs0014"></a>Compilerfehler CS0014

Die erforderliche Datei "file" konnte nicht gefunden werden  
  
 Eine Datei wird vom Compiler benötigt, aber sie ist nicht auf dem System. Stellen Sie sicher, dass der Pfad korrekt ist. Wenn es sich bei der Datei um eine Visual Studio-Systemdatei handelt, müssen Sie möglicherweise die Installation reparieren oder Visual Studio vollständig entfernen und neu installieren.

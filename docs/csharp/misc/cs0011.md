## <a name="example"></a><span data-ttu-id="7a208-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="7a208-108">Example</span></span>  
  
```csharp

{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="7a208-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="7a208-109">Example</span></span>  

 <span data-ttu-id="7a208-110">Die zweite Datei erstellt eine DLL, die eine `C` -Klasse definiert, die von der `B` -Klasse abgeleitet wird, welche im vorherigen Beispiel generiert wurde.</span><span class="sxs-lookup"><span data-stu-id="7a208-110">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="7a208-111">Beispiel</span><span class="sxs-lookup"><span data-stu-id="7a208-111">Example</span></span>

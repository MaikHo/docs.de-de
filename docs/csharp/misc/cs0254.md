# <a name="compiler-error-cs0254"></a>Compilerfehler CS0254
Die rechte Seite einer fixed-Anweisungszuweisung darf kein Umwandlungsausdruck sein.  
  
 Auf der rechten Seite eines [fixed](../language-reference/keywords/fixed-statement.md) -Ausdrucks darf keine Umwandlung verwendet werden. Weitere Informationen finden Sie unter [Unsicherer Code und Zeiger](../programming-guide/unsafe-code-pointers/index.md).  
  
 Im folgenden Beispiel wird CS0254 generiert:  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```

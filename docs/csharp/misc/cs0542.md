# <a name="compiler-error-cs0542"></a><span data-ttu-id="515d7-103">Compilerfehler CS0542</span><span class="sxs-lookup"><span data-stu-id="515d7-103">Compiler Error CS0542</span></span>

<span data-ttu-id="515d7-104">"Benutzerdefinierter Typ": Membernamen dürfen nicht dem einschließenden Typ entsprechen.</span><span class="sxs-lookup"><span data-stu-id="515d7-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="515d7-105">Die Member einer Klasse oder Struktur dürfen nicht den gleichen Namen wie die Klasse oder Struktur haben, es sei denn, der Member ist ein Konstruktor.</span><span class="sxs-lookup"><span data-stu-id="515d7-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="515d7-106">Im folgenden Beispiel wird CS0542 generiert:</span><span class="sxs-lookup"><span data-stu-id="515d7-106">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="515d7-107">Dieser Fehler kann verursacht werden, wenn Sie versehentlich einen Rückgabetyp für einen Konstruktor festlegen, der ihn tatsächlich in eine normale Methode umwandelt.</span><span class="sxs-lookup"><span data-stu-id="515d7-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="515d7-108">Im folgenden Beispiel wird CS0542 generiert, weil `F` eine Methode und kein Konstruktor ist, da sie über einen Rückgabetyp verfügt:</span><span class="sxs-lookup"><span data-stu-id="515d7-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="515d7-109">Wenn Ihre Klasse den Namen "Element" hat und über einen als `this`deklarierten Indexer verfügt, kann dieser Fehler ausgegeben werden.</span><span class="sxs-lookup"><span data-stu-id="515d7-109">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="515d7-110">Ein Standardindexer erhält im ausgegebenen Code den Namen "Element", wodurch der Konflikt entsteht.</span><span class="sxs-lookup"><span data-stu-id="515d7-110">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>

```csharp  
{  
   public int this[int i]  // CS0542  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```

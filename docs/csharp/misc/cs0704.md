## <a name="example"></a><span data-ttu-id="6f563-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6f563-107">Example</span></span>

```csharp  
// CS0704.cs

{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}

{  
    public static void Main() {}  
}
```

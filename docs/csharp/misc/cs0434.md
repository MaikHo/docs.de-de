## <a name="example"></a><span data-ttu-id="99934-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="99934-109">Example</span></span>

```csharp  
{  
    namespace NsImpAggPubImp
    {  
        public class X { }  
    }

```

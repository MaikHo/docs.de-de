 <span data-ttu-id="5b45b-157">Im folgenden Beispiel wird gezeigt, wie Elemente verwendet werden, **\<add>** um die Listener und der Listener-Auflistung hinzuzufügen `MyListener` `MyEventListener` . **Listeners**</span><span class="sxs-lookup"><span data-stu-id="5b45b-157">The following example shows how to use **\<add>** elements to add the listeners `MyListener` and `MyEventListener` to the **Listeners** collection.</span></span> <span data-ttu-id="5b45b-158">`MyListener` erstellt eine Datei `MyListener.log` mit dem Namen und schreibt die Ausgabe in die Datei.</span><span class="sxs-lookup"><span data-stu-id="5b45b-158">`MyListener` creates a file called `MyListener.log` and writes the output to the file.</span></span> <span data-ttu-id="5b45b-159">`MyEventListener` erstellt einen Eintrag im Ereignisprotokoll.</span><span class="sxs-lookup"><span data-stu-id="5b45b-159">`MyEventListener` creates an entry in the event log.</span></span>
